A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Makup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Até este momento, vimos os seguintes recursos da linguagem:

Recurso Tags Associadas Descrição
Links <a> Criar links entre páginas.
Listas <ol> e <li> Criar listas de itens ordenados.
<ul> e <li> Criar listas de itens não ordenados.
Imagens <img> Inserir imagens.
